# Game Member Generator

スポーツ（テニスやバドミントンなど）の練習会で、参加メンバを元に自動で試合の組み合わせ（マッチメイキング）を行う Flutter アプリケーションです。

## 特徴

### 1. Material UI (Material 3)

- **遠隔視認性**: コートから離れた場所に置いたデバイスでも状況が把握できるよう、極太フォントと巨大なカードデザインを採用しています。
- **レスポンシブ・グリッド**: 画面幅や試合数に応じて、3面横並びや2×2グリッドなど、最適なレイアウトに自動調整されます。
- **ダークモード対応**: 屋内外の環境に合わせて目に優しい配色を提供します。

### 2. インテリジェント・マッチメイキング

過去の全試合履歴を数学的に解析し、公平で新鮮な組み合わせを瞬時に算出します。

- **優先評価基準**:
  1. **公平な休憩回数**: 休憩回数が均等になるようにマッチング
  2. **種目バランス**: 男子/女子/混合ダブルスの出場機会を均等化。
  3. **ペアの均等化**: 同じ人とばかり組まないよう、未結成ペアを優先。
  4. **対戦相手の均等化**: 未対戦相手とのマッチを優先。
- **同時出場制限 (Excluded Partners)**: 子連れ夫婦や、常に一方が子守をする必要があるペアなど、「同じ回に絶対に出さない」組み合わせを設定可能。

### 3. 柔軟な試合管理

- **直感的な選手交代 (Swap)**: プレイヤーを長押し＆タップするだけで、試合中・休憩中を問わず手動で入れ替えが可能。
- **試合設定**: コートごとに「男子W」「混合W」などの形式を自由に指定して生成できます。
- **詳細な統計**: 各プレイヤーの出場回数、種目内訳、ペア回数などが自動集計され、公平性をいつでも確認できます。

### 4. メンバ管理とデータ共有

- **簡単登録**: 名前と「よみがな」による正確な五十音順ソート。
- **インポート/エクスポート**: クリップボード経由、またはJSON/CSVファイルによるメンバリストの共有に対応。機種変更やグループ間での共有がスムーズです。
- **SQLite 永続化**: ネットワーク不要で、すべてのデータは端末内に安全に保存されます。

## 技術スタック

- **Framework**: Flutter (Material 3)
- **State Management**: ChangeNotifier & AnimatedBuilder (宣言的UI最適化)
- **Database**: sqflite (FFI対応。Android/iOS/Linux/Windowsに対応)
- **Architecture**: Domain-Driven Design (Clean Architecture 的なレイヤ分割)
  - `Domain`: 純粋なビジネスロジックとエンティティ
  - `Infrastructure`: 外部ストレージ（SQLite）との接続
  - `Presentation`: 状態管理（Notifiers）とレスポンシブUI

## 開発環境のセットアップ (Linux)

Linux環境で実行する場合、SQLite (FFI) を動作させるために以下のパッケージが必要です。

```bash
sudo apt-get update
sudo apt-get install libsqlite3-dev
```

## テスト

アルゴリズムの公平性や、データ保存の型安全性を担保するためのテストが実装されています。

```bash
flutter test
```
