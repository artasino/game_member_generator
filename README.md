# 🏸 Game Member Generator
## 概要
### 「次は誰だっけ？」をゼロに。サークル運営をもっと楽に

スポーツ（テニス・バドミントン・卓球など）の練習会で、公平で楽しい試合の組み合わせを自動作成するFlutterアプリです。  
「コートごとに種目を決めて自動生成したい」「休憩の回数がバラバラ」「いつも同じ人と組んでしまう」「子守りで片方は休ませたい」といった**練習運営のリアルな悩み**を、数学的なロジックで解決します。

---

## 練習運営者の「助かる！」が詰まった キー機能

### 1. 試合形式の「自動提案」と「自由設定」

参加メンバーの男女比やこれまでの出場履歴を分析し、最適な試合形式（男子ダブルス、女子ダブルス、混合ダブルス）をアプリが自動で提案します。

* **男女の出場機会を平滑化**: 特定の性別だけが休みすぎないよう、最適なバランスを算出。
* **コートごとの個別設定**: 「1番コートは男子、2番コートは混合」といった手動でのこだわり設定ももちろん可能です。

### 2. 公平なマッチメイキング（均等化）
過去の全試合をアプリが記憶し、以下の優先順位で「全員が納得する」組み合わせを瞬時に計算します。
* **休憩回数の平等**: 特定の人だけが休みすぎないように調整。
* **ペア・対戦相手のバラけ**: 「またこの人とペア？」というマンネリを防ぎ、交流を促進します。
* **種目バランス**: 男子W、女子W、混合Wの出場機会を均等にします。

### 3. 「子守りペア」の同時出場を強力ブロック
「夫婦で参加しているけれど、一方は必ず子供を見ていなければならない」というケースに完全対応。
* **特定ペアの同時出場禁止**: 指定した2人が同じタイミングでコートに入らないよう、アプリが自動で片方を休憩に回します。
* **視覚的なフィードバック**: 同時出場制限で休んでいるプレイヤーには、休憩エリアで哺乳瓶アイコンが表示されます。

### 4. 費用計算・カンタン精算

練習会終わりの面倒な集金計算を自動化します。

* **シャトル・ボールの消耗計算**: 予め単価を登録しておき、当日使った個数を入力するだけで費用を算出。
* **コート代などの合算**: その他の雑費も追加して、1人あたりの支払額をパパッと算出。
* **分担対象の切り替え**: 「男子のみ」「女子のみ」「全員」など、費用ごとの負担対象を柔軟に設定可能。

### 5. 「見える化」と「自由な編集」
「誰が何回出たか」「このペアは何回目か」を常に集計して表示。さらに、リアルタイム編集にも柔軟に対応できます。
* **統計表示**: 出場回数やペア回数が一目でわかるため、公平性を客観的に示せます。
* **直感的な選手交代（スワップ）**: 「急に足が痛くなった」という時も、画面上の選手を長押しして入れ替えるだけ。

## 📱 現場で使いやすいデザイン (Material 3)

* **とにかく文字が大きい**: 遠くでも自分の名前が見えるよう、視認性を重視したフォントとカードデザインを採用。
* **レスポンシブ対応**: スマホ・タブレット・PC、どのデバイスでも画面サイズに合わせてコートの並びが自動最適化されます。

## 🚀 使い方

### 準備
1. メンバーを登録（名前とよみがなで自動整列）。
2. 過去のリストをJSONやCSVからサクッとインポートすることも可能です。

### 試合開始
1. 今日参加するメンバーにチェックを入れる。
2. 「組み合わせ生成」ボタンを押す。
3. 試合が終わったら「終了」を押して次へ！
4. もし気に入らなかったら、手動編集or再生成ボタン！

### 精算

1. 全ての試合が終わったら「費用計算」タブへ。
2. 使用したシャトル数やコート代を入力して、1人あたりの金額を確認！

## 🔄 Firebase Hosting デプロイ自動化（GitHub Actions）

`main` 向けPRが **merge（closed+merged）** された時（または手動実行）に、次を自動実行します。

1. `flutter pub get`
2. `flutter build web --release`
3. `firebase deploy --only hosting`

ワークフロー: `.github/workflows/deploy_firebase_hosting.yml`

### 事前設定（GitHub Secrets）

リポジトリの **Settings > Secrets and variables > Actions** に、以下の Secret を追加してください。

- `FIREBASE_SERVICE_ACCOUNT_GAME_MEMBER_GENERATOR`
  - Firebase プロジェクト `game-member-generator` で発行したサービスアカウント JSON

> サービスアカウントには Hosting へデプロイ可能な権限（例: Firebase Admin もしくは必要最小限のカスタム権限）を付与してください。
